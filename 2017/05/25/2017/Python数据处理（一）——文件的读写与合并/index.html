好了，闲话少说，这解决这个问题之前，首先要明确的一个问题——文件的路径。

对于文件，我们有两种方式进行访问，一种是绝对路径，一种是相对路径。那么什么是绝对路径？就是路径中带有盘符的，形如D:\task_1\data\1\result1.csv的路径。至于相对路径，则是从当前路径开始的路径。假如现在的路径是D:\task_1\，相对路径就可以简写成data\result.csv。这样是不是少了一大截。而且相对路径所带来的好处还不止这么多！我们这个文件夹现在在D盘，但是由于时间有限，我们将其Copy到了U盘中，利用笔记本来运行以达到节省时间的目的。但是U盘的盘符可不一定是D，用绝对路径的访问将可能访问到空目录，那时就只能傻眼了。但是，如果使用相对路径，那么不管放在哪里，只要文件夹内部文件不变，那就可以随时访问了。好了，说了这么多，其实就像表达一个内容，那就是相对路径太好了。
